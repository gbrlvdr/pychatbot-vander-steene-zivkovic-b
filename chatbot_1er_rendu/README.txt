## Fonctionnalités principales

Ouvrir le fichier "main.py". L'autre fichier ne contient que les fonctions. 

Ce programme effectue une analyse des discours de différents présidents français à partir de fichiers texte. Voici les principales fonctionnalités :

1. **Liste des mots les moins importants :** Affiche une liste de mots dont le score TF-IDF est proche de zéro. Ces mots peuvent être considérés comme moins importants dans le corpus de documents.

2. **Mot avec le score TF-IDF le plus élevé :** Affiche le mot ayant le score TF-IDF le plus élevé dans l'ensemble des discours.

3. **Mot le plus répété par un président :** Permet à l'utilisateur de saisir le nom d'un président et affiche le mot le plus répété dans ses discours.

4. **Mot « Nation » :** Indique le président qui a le plus souvent évoqué le mot « Nation » et fournit une liste des présidents qui ont également parlé de ce mot.

5. **Premier président à parler du climat/écologie :** (Fonctionnalité en cours de développement)

6. **Mots évoqués par tous les présidents :** Affiche les mots qui ont été évoqués par tous les présidents dans leurs discours.
